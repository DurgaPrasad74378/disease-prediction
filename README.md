# disease-prediction
Multi-class disease prediction from symptoms using scikit-learn, XGBoost &amp; Neural Networks | Python · ML · Healthcare AI

## 🚀 Overview
The system allows users to select multiple symptoms from a standardized list. Behind the scenes, the application:
1.  **Preprocesses** the input by mapping text symptoms to numerical severity weights.
2.  **Analyzes** the data using a trained Random Forest model.
3.  **Predicts** the most likely diagnosis and displays it to the user.
